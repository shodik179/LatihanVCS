# langkah langkah penggunaan git
1. git clone "isi link"
2. git add "latihan vcs"(nama file yang mau di eksekusi)
3. git add "README.md"
4. git status (untuk melihat file)
5. git add . (Untuk mengeksekusi semua file)
6. git commit -m "menambahkan file repository"
7. git push -u origin main (push pada github)
